# Инструкция по обновлению сайта

## 🚀 Шаги для обновления дизайна на сайте

### Шаг 1: Добавьте изображение
Сохраните ваше изображение офиса в папку:
```
public/images/hero-bg.jpg
```

### Шаг 2: Проверьте локально
Запустите проект локально, чтобы убедиться, что всё работает:
```bash
npm run dev
```
Откройте http://localhost:3000 и проверьте новый дизайн.

### Шаг 3: Закоммитьте изменения
```bash
git add .
git commit -m "Обновление дизайна в стиле Gett"
```

### Шаг 4: Отправьте на сервер
```bash
git push origin main
```

### Шаг 5: Деплой (зависит от вашего хостинга)

#### Если используете Vercel:
1. Изменения автоматически задеплоятся после push
2. Проверьте статус на https://vercel.com/dashboard
3. Обычно деплой занимает 1-2 минуты

#### Если используете другой хостинг:
```bash
# Соберите проект
npm run build

# Загрузите файлы из папки .next на сервер
# или используйте команду деплоя вашего хостинга
```

## 🔧 Если что-то пошло не так

### Проблема: Изображение не отображается
**Решение:** Убедитесь, что файл находится по пути `public/images/hero-bg.jpg`

### Проблема: Старые стили всё ещё видны
**Решение:** Очистите кеш браузера (Ctrl+Shift+R) или кеш сборки:
```bash
rm -rf .next
npm run dev
```

### Проблема: Ошибки при сборке
**Решение:** Переустановите зависимости:
```bash
rm -rf node_modules
npm install
npm run build
```
