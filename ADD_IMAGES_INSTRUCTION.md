# Инструкция по добавлению изображений

## 📸 Изображения для добавления

У вас есть 2 отличных фотографии офиса. Их нужно сохранить в папку `public/images/`

## 📁 Куда сохранить:

### Изображение 1: Встреча в офисе (мужчина с ручкой)
**Путь:** `public/images/office-meeting.jpg`
- Это первое изображение с деловой встречей
- Используется в шагах 1 и 3 секции "Как это работает"

### Изображение 2: Рабочее место (женщина за ноутбуком)
**Путь:** `public/images/office-workspace.jpg`
- Это второе изображение с рабочим местом
- Используется в шаге 2 секции "Как это работает"

## 🔧 Как добавить:

### Способ 1: Через проводник Windows
1. Откройте папку проекта: `c:\formai-landing\public\images\`
2. Скопируйте туда оба изображения
3. Переименуйте их:
   - Первое фото → `office-meeting.jpg`
   - Второе фото → `office-workspace.jpg`

### Способ 2: Через VS Code
1. В VS Code откройте папку `public/images/`
2. Перетащите изображения в эту папку
3. Переименуйте их соответственно

## ✅ Проверка

После добавления изображений:
1. Сервер разработки автоматически обновится
2. Откройте http://localhost:3000
3. Прокрутите до секции "Как это работает"
4. Вы должны увидеть ваши фотографии в каждом шаге

## 📊 Структура должна быть:

```
formai-landing/
├── public/
│   └── images/
│       ├── Professionals_talking_in_202603271314.webp  ← Уже есть (hero)
│       ├── office-meeting.jpg                          ← Добавить
│       └── office-workspace.jpg                        ← Добавить
```

## 🚀 После добавления

Когда изображения будут на месте:
1. Проверьте локально на http://localhost:3000
2. Закоммитьте изменения:
   ```bash
   git add public/images/
   git commit -m "Добавление изображений для секции Как это работает"
   git push origin main
   ```
3. Vercel автоматически задеплоит обновления

## ⚠️ Важно

- Имена файлов должны быть **точно** такими: `office-meeting.jpg` и `office-workspace.jpg`
- Формат: JPG или JPEG
- Рекомендуемый размер: до 500 КБ каждое
